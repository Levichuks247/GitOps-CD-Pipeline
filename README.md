# GitOps-CD-Pipeline

## 🚀 Overview
This project demonstrates a **Modern GitOps Workflow** using Amazon EKS and ArgoCD. 

Unlike traditional "Push" CI/CD models, this pipeline uses a **"Pull"** architecture where the cluster automatically synchronizes its state with the manifests stored in this repository.

### 🛠 Tech Stack
* **Cloud:** AWS (EKS, VPC, IAM)
* **IaC:** Terraform
* **GitOps:** ArgoCD
* **Orchestration:** Kubernetes
* **CI/CD:** GitHub Actions (Infrastructure Provisioning)
